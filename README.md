# LocalLLM.Ret-Aug-Assist
### Trim unnecessary parts of PDF and EPUB before providing them to your LLM retrieval augmented system
- Tools to trim PDF documents and EPUB within a given page range, 
- A lot of businesses/organizations/individuals are uploading things like entire PDFs or EPUBs into their LLM retrieval augmented system without trimming out the irrelevant parts. 
    + For example, the '*introduction*', '*about the author*', '*index*', and other prefacing or ending sections.

- Remove the irrelevant sections you're giving them.
- Includes a utility to convert EPUB to PDF.
